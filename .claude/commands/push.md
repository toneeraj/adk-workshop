Commit and push all changes to GitHub.

## Task

1. Check git status for changes
2. Stage all changes: `git add .`
3. Create a commit with a descriptive message based on the changes
4. Push to origin: `git push`

## Constraints

- Write a clear, concise commit message describing what changed
- If there are no changes, inform me and do nothing
- If the repo isn't initialized, run `git init` first
- If no remote is set, add origin: `git remote add origin git@github.com:toneeraj/adk-workshop.git`
